Что такое промис?

Промис (Promise) — это объект, который представляет результат асинхронной операции. Он может находиться в трёх состояниях:

  • pending — операция ещё выполняется;
  • fulfilled — операция успешно завершена, есть результат;
  • rejected — операция завершилась с ошибкой.

Вместо того чтобы ждать завершения операции, код «подписывается» на её результат с помощью методов .then() и .catch(), либо через async/await. Это позволяет писать асинхронный код так, словно он последовательный.

Примеры

JavaScript: создание и использование промиса


// Создаём промис
function asyncTask() {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      const success = true;
      if (success) {
        resolve("Задача выполнена!");
      } else {
        reject("Произошла ошибка");
      }
    }, 1000);
  });
}

// Используем через then/catch
asyncTask()
  .then(result => console.log("Результат:", result))
  .catch(error => console.error("Ошибка:", error));

JavaScript: async/await


async function run() {
  try {
    const result = await asyncTask();
    console.log("Результат через await:", result);
  } catch (err) {
    console.error("Ошибка:", err);
  }
}

run();

Python (аналог через asyncio)


import asyncio

async def async_task():
    await asyncio.sleep(1)
    return "Задача выполнена!"

async def main():
    try:
        result = await async_task()
        print("Результат:", result)
    except Exception as e:
        print("Ошибка:", e)

asyncio.run(main())
