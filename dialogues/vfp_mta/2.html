Что такое асинхронное программирование?

Асинхронное программирование — это подход, при котором программа не блокируется в ожидании долгих операций (например, чтения с диска, сетевых запросов или работы с базой данных). Вместо этого задача «ставится в очередь», и программа продолжает выполнять другие действия.

Когда операция завершается, вызывается обработчик (callback), либо срабатывает promise, либо возвращается результат через ключевое слово await (в языках, где это есть).

Главная цель — эффективнее использовать время процессора и сделать программы отзывчивыми даже при большом числе задач ввода-вывода.

Примеры

JavaScript (Node.js)


// Асинхронное чтение файла
const fs = require('fs').promises;

async function readFile() {
  try {
    let data = await fs.readFile('data.txt', 'utf8');
    console.log("Содержимое файла:", data);
  } catch (err) {
    console.error("Ошибка чтения:", err);
  }
}

readFile();
console.log("Этот вывод идёт раньше чтения файла!");

Python (asyncio)


import asyncio

async def fetch_data():
    print("Запрос данных...")
    await asyncio.sleep(2)  # имитация долгой операции
    print("Данные получены")
    return {"value": 42}

async def main():
    task = asyncio.create_task(fetch_data())
    print("Делаем что-то ещё...")
    result = await task
    print("Результат:", result)

asyncio.run(main())

C# (.NET)


using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        Console.WriteLine("Начинаем работу...");
        string text = await File.ReadAllTextAsync("data.txt");
        Console.WriteLine("Содержимое файла: " + text);
        Console.WriteLine("Работа завершена.");
    }
}
